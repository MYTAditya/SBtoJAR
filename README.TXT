SBtoJAR (Scratch to JAR) is a program to convert Scratch 1.4 SB files to Java JAR files.
This Program is originally developed by gbear605 which is available on <https://github.com/gbear605/ScratchToJAR>.
The BlueJ port is by Mastered YT Aditya with some changes to texts which is available on <https://github.com/MYTAditya/SBtoJAR>.
